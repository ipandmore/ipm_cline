Introduction

.. _what-it-does:

What does it do?
================

Add copyright lines to your TYPO3 project. With the "today"-Feature the plugin keeps track of the current date in the copyright line, so you won't have to update the date at New Year.

- Easy to install
- Easy to use
- FlexForm only
